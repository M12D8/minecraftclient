BUILD PROCESS
================================================================================

The updated build.bat now:

1. Locates Visual Studio C++ compiler
2. Compiles all ImGui source files:
   - imgui.cpp (core)
   - imgui_draw.cpp (rendering)
   - imgui_widgets.cpp (controls)
   - imgui_tables.cpp (tables)
   - imgui_impl_dx11.cpp (DirectX backend)
   - imgui_impl_win32.cpp (Windows backend)

3. Compiles your code:
   - main_imgui.cpp (main DLL)
   - imgui_overlay.cpp (ImGui wrapper)

4. Links against:
   - user32.lib (Windows UI)
   - gdi32.lib (Graphics)
   - d3d11.lib (DirectX 11)
   - dxgi.lib (Display)

5. Produces:
   - build/esp_mod.dll (~1-2 MB)
   - build/injector.exe (~100-300 KB)

Compile time: ~30-60 seconds (depending on system)

CUSTOMIZATION
================================================================================

The implementation is designed for easy customization:

ADD A CHECKBOX:
  static volatile bool g_myFeature = false;
  ImGui::Checkbox("My Feature", (bool*)&g_myFeature);

ADD A SLIDER:
  static float g_value = 5.0f;
  ImGui::SliderFloat("Value", &g_value, 0.0f, 10.0f);

ADD A NEW PANEL:
  ImGui::SetNextWindowPos(...);
  if (ImGui::Begin("MY PANEL", ...)) {
    ImGui::Checkbox(...);
    ImGui::End();
  }

CHANGE COLORS:
  colors[ImGuiCol_WindowBg] = ImVec4(R/255.f, G/255.f, B/255.f, A);

See IMGUI_QUICK_START.md for detailed examples.
