## 🚀 Quick Start (No Local Building!)

### Step 1: Create GitHub Account (If Needed)
```
https://github.com/join → Create account → 2 minutes
```

### Step 2: Create Repository
```
https://github.com/new → Name it → Create → 1 minute
```

### Step 3: Push Code
```bash
cd C:\Users\Rdp1\Desktop\Mincraft

git init
git add .
git commit -m "Initial: Vape v4 Fabric Mod"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/minecraft-esp-mod.git
git push -u origin main
```
(2 minutes)

### Step 4: Watch Build
```
Go to https://github.com/YOUR_USERNAME/minecraft-esp-mod/actions
→ Wait for green ✅ checkmark
→ Takes ~5-10 minutes first time
```

### Step 5: Download JAR
```
Click on successful build
→ Scroll to Artifacts
→ Download espmod-jar.zip
→ Extract espmod-1.0.0.jar
```

### Step 6: Install & Play
```bash
copy espmod-1.0.0.jar %APPDATA%\.minecraft\mods\

# Play!
# Press INSERT in-game
```
